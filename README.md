# MATLAB scripts for "A Bayesian Analysis of the Radioactive Releases of Fukushima" (AISTATS 2012)

## Instructions
1. Start MATLAB
1. Set opengl to software (avoids unexpected MATLAB crash)

 ```matlab
 opengl software;
 ```
1. Run the experiment

 ```matlab
 s_infer_fukushima1;
 ```
